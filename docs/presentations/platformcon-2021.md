# Making Deployments Boring

## PlatformCon 2021 Talk Outline

### The Problem
- Deployments are exciting (in a bad way)
- Rollbacks at 3 AM
- Heroic firefighting culture

### The Solution
- Safety gates before production
- Automated canary analysis
- Gradual rollouts

### Key Metrics
- Rollback rate: 15% → 2%
- MTTR: 45min → 12min
- On-call pages: 8/week → 2/week

### Takeaways
1. Invest in prevention, not just detection
2. Make safe defaults easy
3. Measure deployment health, not just system health
